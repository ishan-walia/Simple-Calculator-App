# Enhanced Calculator App

A feature-rich calculator application written in Kotlin. Includes:
- User authentication (login/register)
- Basic and scientific calculator functions
- Unit conversion
- CGPA calculator
- Calculation history
- Dark mode support
- Memory functions
- Error handling for division by zero
